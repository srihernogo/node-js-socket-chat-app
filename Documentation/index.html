Introduction

Chatvia is a fully featured Node.js Responsive Chat App Template with MongoDB. Chatvia is built with Bootstrap v4.5.0, Node.js, CSS and responsive with all devices and supported with Dark, Light modes. You can change mode very quickly by doing a couple of changes.It has many features like one to one chat, group chat, contact list, send files, online users, read and unread new messages from users, authentication pages and many more.
